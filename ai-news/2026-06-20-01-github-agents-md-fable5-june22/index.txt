**GitHub Copilot: two June 17–18 updates.** Copilot code review now reads an `AGENTS.md` file in your repo root — the Copilot equivalent of Claude Code's `CLAUDE.md`. Add it to give the code review agent standing project instructions: which patterns to flag, which paths are sensitive, team-specific review conventions. It loads automatically on every review request, no per-request prompt needed. The same update improves the UI for requesting Copilot agent reviews on draft PRs directly. Separately, **auto mode in Copilot Chat** — which selects the best model for each request without manual configuration — is now generally available for all users after being opt-in.

**Fable 5: the June 20 window closed, June 22 is next.** Prediction markets had a "before June 20" contract for Fable 5 restoration — that window expired today with both models still offline. Kalshi's July 1 market sits at 57% (down from 68% earlier this week). The Pro/Max refund window also closes today. **June 22** is the next hard milestone: Anthropic's scheduled in-person meeting with White House officials, working toward a framework to restore access starting with US customers. Watch [anthropic.com/news](https://www.anthropic.com/news/fable-mythos-access) for the authoritative signal.

**Why it matters:** AGENTS.md lets Copilot code review carry your team's conventions into every PR — not just generic linting. On Fable 5: the negotiating window is narrowing; the June 22 meeting is the first formal venue where a restoration framework could actually be agreed.
