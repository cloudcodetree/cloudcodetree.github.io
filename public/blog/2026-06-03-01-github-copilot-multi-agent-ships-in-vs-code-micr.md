At Build 2026, Microsoft launched multi-agent support for VS Code, letting an orchestrator agent spawn parallel subagents for simultaneous workstreams — linting, test generation, documentation, and security review can all run concurrently instead of sequentially. Available now in preview for Copilot subscribers; the GitHub Copilot native desktop app also dropped in preview.
