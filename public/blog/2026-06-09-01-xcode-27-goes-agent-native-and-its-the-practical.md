Following the keynote (June 8), the developer story worth your attention is Xcode 27's agentic coding overhaul: you can now run Anthropic, Google, and OpenAI agents directly inside Xcode, with interactive planning, multi-turn Q&A, and a canvas that renders Markdown, diffs, and previews side-by-side. Critically, Xcode now gives agents the tools to *validate their own work* — writing and running tests, trying ideas in isolated Playgrounds, checking visual changes via previews, and driving the simulator through a new Device Hub — so they run autonomously for longer. Plug-ins let you add custom skills, wire in tools over **MCP**, and connect any agent that speaks the **Agent Client Protocol (ACP)**; GitHub and Figma ship one-click integrations. *New since the June 8 briefing, which covered Siri/Gemini and iOS 27 third-party assistants but not the Xcode/dev-tooling side.*
