A well-maintained `.cursorrules` / `CLAUDE.md` is described as one of the highest-leverage investments in an AI workflow: it encodes architecture, conventions, and constraints so a generic assistant produces code that fits your codebase. Pair it with a short "how we use AI here" doc with examples of AI-assisted PRs that passed review to reduce team variance.
