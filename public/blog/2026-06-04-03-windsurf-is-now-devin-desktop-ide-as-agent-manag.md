On June 2, Cognition retired the Windsurf brand and relaunched as Devin Desktop, with a Rust-rewritten engine (Devin Local, 30% more token-efficient), Agent Client Protocol (ACP) support, and a new default screen that's a Kanban board of your running agents rather than a code editor. The paradigm shift: you're now a "goal-setter and reviewer" rather than a line-level coder. If you're on Windsurf, the update arrives as an OTA update with no pricing changes. Cascade is deprecated July 1.
