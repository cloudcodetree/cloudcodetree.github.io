Starting August 2026, GitHub Copilot's default engine switches from GPT-4 Turbo to Project Polaris, Microsoft's own in-house coding model. This ends Copilot's dependence on OpenAI and gives Microsoft full control over the model roadmap. Separately, the Agent Framework for .NET and Python hit production-ready 1.0 and is now Microsoft's official standard for multi-agent systems on Azure.
